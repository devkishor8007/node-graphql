# node-graphql
sample project with node.js and graphql

## Use dependencies 
    "apollo-server": "^3.10.2",
    "apollo-server-core": "^3.10.2",
    "graphql": "^16.6.0"
    
## To Use this Repository
     git clone https://github.com/devkishor8007/node-graphql.git
     cd node-graphql
     npm install
     node index.js
